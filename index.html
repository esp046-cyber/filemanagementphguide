File Compression

‚ñº

How to Zip/Unzip Files

  • Windows: Right-click ‚Üí Send to ‚Üí Compressed folder
  • Mac: Right-click ‚Üí Compress
  • Android: Use RAR app or Files by Google
  • iOS: Files app ‚Üí Select ‚Üí More ‚Üí Compress

Recommended Tools

  • 7-Zip: Free, open-source, highest compression
  • WinRAR: Popular, supports many formats
  • PeaZip: Free alternative, user-friendly
  • Online: ezyZip, CloudConvert (no installation)

When to Compress

  • YES: Multiple files na isasama (reports, photos)
  • YES: Large text documents, spreadsheets
  • YES: Email attachments over 10MB
  • NO: Already compressed (JPG, MP3, MP4, PDF)

Compression Tips

  • Use .zip format para universal compatibility
  • Split large files: 7-Zip ‚Üí Split to volumes
  • Password protect: Add password before compressing
  • Compression ratio: Higher = smaller pero slower

üí° Tip: Text files at Excel may mataas na compression ratio (50-90%). Videos at photos minimal lang (5-10%) kasi compressed na sila.

Troubleshooting

‚ñº

Sync Issues

  • Check internet: Stable ba ang connection?
  • Restart app: Close completely then reopen
  • Clear cache: Settings ‚Üí Clear cache & data
  • Reinstall: Last resort, uninstall then install ulit

Upload/Download Problems

  • File too large: Compress o split into smaller parts
  • Slow upload: Check upload speed, try WiFi instead of mobile data
  • Failed upload: Retry, check file name (walang special chars)
  • Stuck download: Pause then resume, or cancel at restart

Storage Full Solutions

  • Delete duplicates: Use built-in duplicate finder
  • Empty trash: Permanently delete old files
  • Move to external: Transfer to hard drive o USB
  • Upgrade plan: Consider paid storage kung kailangan

Cannot Open File

  • Wrong app: Install proper application (PDF reader, Office)
  • Corrupted file: Request sender to resend
  • Version mismatch: Update app to latest version
  • Permission denied: Request access from owner

üí° Tip: Most sync issues ay nare-resolve by signing out then signing back in. Second option: Check if may update ang app.

File Naming Best Practices

‚ñº

Standard Naming Convention

  • Format: YYYY-MM-DD_ProjectName_Description_v#
  • Example: 2026-01-08_ClientReport_Budget_v3.xlsx
  • Use underscores: Not spaces (compatibility issues)
  • Be descriptive: "Budget2026" not "file123"

Version Control Naming

  • v1, v2, v3: Major revisions
  • v1.1, v1.2: Minor edits
  • DRAFT: Work in progress
  • FINAL: Approved version (avoid FINAL_FINAL!)

Industry-Specific Conventions

  • Design: ProjectName_AssetType_Size (Logo_PNG_1080x1080)
  • Marketing: Campaign_Platform_Date (BlackFriday_FB_2026-11)
  • Legal: DocType_ClientName_Date (Contract_AcmeCorp_2026-01)
  • Photos: Event_Location_Date (Wedding_Tagaytay_2026-01-08)

Why YYYY-MM-DD is Best

  • Auto-sort: Chronological order automatically
  • ISO standard: International format, no confusion
  • Future-proof: Works for any date range
  • Searchable: Easy to find by date

üí° Tip: Avoid "FINAL", "FINAL2", "FINAL_REALLY". Use v1, v2, v3 instead. Pag dating ng "FINAL_FINAL_REVISED", malito na lahat!
